はじめまして、こんにちは。畑優と申します。
京都府亀岡市の出身です。亀岡市は「霧の都 亀岡」と言われるほど、冬は霧が深くなります。 100m先が見えないくらい濃霧になることが多く、ホラーゲームの 『サイレントヒル』や 映画『ミスト』のような世界を体験できます。
「かめおか霧のテラス」と 呼ばれる絶景スポットもあるので、遊びに来てくださいね～。

前職はゲーム開発会社にて、ゲームの開発･検証や、企画提案、それからWebディレクターとして サイト の運営・管理を行っていました。
Webディレクターとは言うものの、コーディングをしていた訳ではなく、リアルイベントに合わせたWebページの企画を考えたり、 既存ページの改修プランを立てたりと、プランナー的な仕事をしていました。
なので、実際にコーディングをやり始めたのは最近です。

現在はWebサービスのリリースを目指して開発をしています。
0からリリースまでの流れ(設計→開発→公開→運用･保守)を経験したいと思い、開発を始めました。知らなかったことばかりで、 毎日ヒーヒー言ってますが、その辛さ以上に、知らない世界を体験できるのが楽しいです。

Like

好きなものは「ボードゲーム」
オススメ４選

Dixit

『一枚の絵は、千の言葉に値する！』が、このボードゲームのキャッチコピーです。 絵がカード全面に描かれていて、文字はありません。そう、このボードゲームは文字が読めなくても遊べます。 なので、小さいお子様から細かい文字の読みにくい高齢の方まで、誰でも一緒に楽しめます。また、「たった一つの正解」というものがなく、 参加したプレイヤーによって「正解」が変わるので、気楽に遊べるゲームです。見ているだけでも絵画鑑賞のように楽しめます。

ブラックストーリーズ

ジャンル「シチュエーションパズル」。水平思考推理ゲームや、yes/noパズルとも言われるものです。 このゲームは大人向けです(ブラックユーモアを多分に含みます)。日本で有名な問いは「ウミガメのスープ」。 「ある男は、レストランで「ウミガメのスープ」を注文し飲みました。男は帰宅し、自殺しました。なぜ？」といったかんじです。 頭を柔らかくして考えるのが好きな人や、思考が閃く瞬間が好きな人にオススメです。 悩みに悩んだ末に閃いたときには、思わず「Eureka!」と叫んでしまうかもしれません。

ツンダール

「シンプル・イズ・ベスト」なボードゲーム。とても簡単なルールで、すぐに遊べます。 このゲームもカードにはイラストのみが描かれていて、文字を必要としません。ゲームシステムも『テトリス』や『ぷよぷよ』のような直感的なものなので、 一回遊んでみるとすぐにゲームシステムを把握できます。運の要素も強いゲームなので、年齢や経験に関係なく遊べます。 トランプと同じくらいの大きさなので、持ち運びやすいところもグッドポイント！

BGA

『ボードゲームアリーナ』。世界最大のオンラインでボードゲームが遊べる神サービスです。 300の国々から参加でき、41の言語に翻訳されています。遊べるゲームは190を超え、300万人以上の登録者がいます(私もその1人)。 最も素晴らしい点が、ブラウザで遊べることです。PC、スマホ、タブレット、PlayStationやXboxなど、 あらゆる環境からでもブラウザを通して遊べます(もちろんクロスプレイで！)。 世界的な「StayHome」の時期には、その影響でサーバがパンクするほど人気でした。興味のある方はお試しあれ。

Now

現在勉強中のスキル

React

触りだして2日。フロントエンドの実装によく使われているので勉強しています。 フロントで処理して動的に動かせる点が面白い。ReactNativeを使えばネイティブアプリの開発もできるので、SPAの開発をやってみたい。

Ruby on Rails

触りだして2週間くらい。どの言語に手を出すか、PythonやPHPと悩みましたが、実際に触ってみて一番書きやすかったのがRubyでした。 フレームワークが便利すぎて、フレームワークなしでは生きられない身体に。

SQLite

Railsのデフォルトデータベース。なので、気づいたら触ってた。軽量で扱いやすいらしいが、 他を知らないので利便性を感じたことはない。フロントエンドがある程度できるようになってからMySQLの勉強を始める予定。

GitHub

ソースコード管理。最近、リモートリポジトリの設定を学び、コードを共有してどこでも開発できるようになった。 最初はGitの設定に苦戦したが、今となってはGit無しとかありえん身体に。複数人での開発の経験もしてみたい。
